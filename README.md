## Featured Scripts
[replay_sfx](https://github.com/junapur/obs-scripts/blob/main/src/replay_sfx.lua) - Plays a sound when you save a replay

<br>

## Installation
1. Download the script you want to use
2. Open OBS Studio
3. Go to **Tools** -> **Scripts**
4. Click the **+** button and select the downloaded `.lua` file
5. Configure settings as needed
